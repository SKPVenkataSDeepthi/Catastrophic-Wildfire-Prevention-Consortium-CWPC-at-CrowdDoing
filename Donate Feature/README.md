## 📌 Overview  
This repository documents the research, analysis, and feature enhancements proposed for the Donate functionality of the Prevent Wildfire platform. The focus is on improving user experience, payment efficiency, and global accessibility.

---

## 🚀 Contributions  

### 1. 💳 Transaction Fee & Multi-Currency Support Findings  

![Transaction Fee Analysis](Images/transaction_fee.png)

- Analyzed existing donation flow and associated transaction costs  
- Identified inefficiencies in payment processing fees across providers  
- Evaluated multi-currency support gaps impacting international donors  

**Proposed:**  
- Optimized payment gateways with lower transaction fees  
- Dynamic currency conversion support  
- Transparent fee breakdown for users  

---

### 2. 📊 Donation Platform Comparative Analytics & Scoring Framework  

![Chatbot](https://raw.githubusercontent.com/SKPVenkataSDeepthi/Catastrophic-Wildfire-Prevention-Consortium-CWPC-at-CrowdDoing/main/Images/chatbot_architecture.png)

- Conducted benchmarking of leading donation platforms  
- Built a structured scoring model based on:
  - Transaction fees  
  - Supported payment methods  
  - Global accessibility  
  - User experience  
  - Security & compliance  

- Developed a comparative analytics framework to:
  - Rank platforms objectively  
  - Support decision-making for integration  
  - Highlight strengths and limitations  

---

### 3. 🤖 Chatbot for Donation Support  

![Tech Stack](https://raw.githubusercontent.com/SKPVenkataSDeepthi/Catastrophic-Wildfire-Prevention-Consortium-CWPC-at-CrowdDoing/main/Images/tech_stack.png)

- Designed and developed a chatbot integrated within the donation flow  

**Key functionalities:**  
- Answer user queries related to payments, currencies, and fees  
- Guide users through the donation process  
- Provide real-time assistance for failed or pending transactions  
- Improve donor confidence and reduce drop-offs  

**Implementation highlights:**  
- Retrieval-Augmented Generation (RAG) based responses  
- Context-aware query handling using NLP techniques  
- Integration with knowledge base for accurate financial guidance
