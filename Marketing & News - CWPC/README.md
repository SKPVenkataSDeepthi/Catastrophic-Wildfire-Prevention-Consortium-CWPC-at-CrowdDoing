# 🔥 Wildfire Intelligence Dashboard

An interactive data analytics dashboard that analyzes wildfire-related news sentiment and identifies high-risk regions across the United States.

---

## 🚀 Live App
👉 https://cwpc-news-sentiment-analysis.streamlit.app

---

## 📊 Project Overview

Wildfires are increasing in frequency and severity, making it critical to monitor risk signals in real time.  

This project uses **news data + sentiment analysis** to:
- Detect wildfire risk trends  
- Identify high-risk states  
- Provide visual insights for decision-making  

---

## 🎯 Key Features

- 📈 **Sentiment Analysis**
  - Positive / Negative / Neutral classification
- 🗺️ **Geographic Risk Mapping**
  - U.S. choropleth map showing wildfire activity
- 📊 **Interactive Visualizations**
  - Pie chart (sentiment distribution)
  - Bar chart (top affected states)
  - Trend line (monthly sentiment trends)
- ⚡ **Real-time Dashboard UI**
  - Clean, modern, and responsive layout
